# 🎨 JJ Art Gallery & Portfolio

Welcome to the official repository of **JJ Art Gallery**. This is a personal portfolio website designed to showcase artistic drawings, paintings, and creative artworks in a beautiful, minimalist digital space.

## 🔗 Live Demo
You can view the live website here:
👉 **[jjartgallery.netlify.app](https://jjartgallery.netlify.app)**

---

## 🛠️ Technologies Used
* **HTML5** - For semantic web structure
* **CSS3** - For modern layout, styling, and design
* **Vanilla JavaScript** - For interactive elements (if applicable)

---

## 🌟 Key Features
* **Multi-Language Support:** Available in 3 languages: **Persian (فارسی)**, **English**, and **German (Deutsch)**.
* **Responsive Design:** Optimized for all screen sizes, including mobile phones, tablets, and desktops.
* **Fast & Lightweight:** Hosted on Netlify with automated GitHub deployments.

---

### 📝 How to run locally
1. Clone or download this repository.
2. Open `index.html` in any web browser.
